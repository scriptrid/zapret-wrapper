### Примечание
Данную поделку следует рассматривать, как игрушку. 
Я ее создал только из-за того, что устал вручную менять конфиги и перезапускать Discord руками во время поиска подходящей
стратегии для обхода.
На данный момент приложение не умеет ровным счетом ничего.

### Описание
Данное изделие позиционируется как обертка над функционалом [zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle), 
которая позволит перебирать стратегии без кучи кликов и ручных перезапусков Discord-а или иных ресурсов. 

### Предварительные требования

- Терпение
- Усидчивость
- Должен быть где-нибудь распакован [zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle)
